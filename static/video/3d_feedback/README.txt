Put all 3D feedback mp4 files in these folders:
- static/video/3d_feedback/colors
- static/video/3d_feedback/family
- static/video/3d_feedback/numbers
- static/video/3d_feedback/relationship
- static/video/3d_feedback/survival

Filename format:
<boy|girl>_<category_prefix>_<sign>.mp4

Examples:
- boy_colors_black.mp4
- girl_family_father.mp4
- boy_number_one.mp4   (numbers uses 'number' in filename)
- girl_relationship_girl.mp4
- boy_survival_yes.mp4
